# Mall of America — Interactive Sales Deck

# Overview

This project presents a premium, browser-based sales deck built around Mall of America, designed to deliver a modern and engaging storytelling experience.

# The platform supports two primary use cases:

Live presentations via screen sharing for sales teams
Self-guided exploration for prospective clients and partners

It aligns closely with key assignment goals, including:

Non-linear and intuitive navigation
High-end visual design
Strong commercial storytelling
Scalable and modular architecture
Lightweight and efficient implementation
Project Rationale

Mall of America was selected as the subject due to its unique combination of:

Large-scale retail ecosystem
Diverse dining and lifestyle offerings
Integrated entertainment attractions
Proximity to hospitality infrastructure
Extensive event-hosting capabilities

These elements make it an ideal case study for demonstrating leasing, sponsorship, and event partnership opportunities within a unified digital experience.

# Technology Stack
HTML, CSS, JavaScript (Vanilla)
Lightweight Node.js static server
No external runtime dependencies

This ensures simplicity, performance efficiency, and ease of deployment.

# Local Setup Instructions
Open a terminal in the project directory
Run: npm start
Navigate to: http://localhost:4173
Project Structure
index.html — Core layout and content structure
styles.css — Visual design system and responsiveness
app.js — Navigation logic, rendering, and interactions
mall-data.js — Content model and structured data
server.mjs — Minimal static server for local hosting
# Design Approach
Premium Visual Identity:
A luxury-inspired design featuring a dark editorial palette, serif typography, glassmorphism elements, and metallic accents.
Non-Linear Navigation:
Sidebar-based navigation with audience-focused pathways for flexible exploration.
Commercial Storytelling:
Content is structured around business outcomes (leasing, partnerships, events) rather than generic descriptions.
Scalable Architecture:
Modular design allows future expansion into deeper, more detailed sections.
Performance Optimization:
Minimal JavaScript, no frameworks, and reduced asset load for fast performance.
# Data Sources

The following official sources were used to ensure accuracy:

Mall of America Press Releases (Visitor statistics, property size, retail data)
Official Events Page (Annual event data)
SEA LIFE Aquarium Page (Attraction details)
Meetings & Events Page (Venue and capacity insights)
Partnership Opportunities Page (Sponsorship positioning)
Additional Notes
The project emphasizes visual storytelling through motion and design, avoiding heavy media assets to maintain performance.
Certain derived metrics (e.g., connected hotel rooms) are calculated from publicly available data.
The architecture is flexible and can be extended into:
A React/Vite application
A version-controlled GitHub repository
A production-ready deployment (e.g., Vercel or Netlify)
# Closing Statement

This project demonstrates the ability to combine design thinking, frontend development, and business storytelling into a cohesive digital experience. It reflects a strong understanding of both user experience design and commercial product presentation, making it suitable for real-world sales and partnership scenarios.
